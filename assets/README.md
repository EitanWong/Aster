# Aster Design Assets

This directory contains all design assets for the Aster project, including logos, icons, and branding guidelines.

## Contents

### Logo Files

- **logo.svg** - Main project logo (200x200px, scalable)
  - Full-color version with gradient
  - Suitable for documentation, branding, and social media
  - Includes glow effects and accent lines

- **favicon.svg** - Favicon version (32x32px, scalable)
  - Simplified star icon
  - Suitable for browser tabs, bookmarks, and app icons
  - Maintains brand identity at small sizes

### Documentation

- **LOGO_GUIDE.md** - Comprehensive logo usage guidelines
  - Design specifications
  - Color palette
  - Usage guidelines (do's and don'ts)
  - Integration examples
  - Future enhancement plans

## Design Philosophy

The Aster logo embodies the project's core values:

1. **Innovation** - Modern, forward-thinking design
2. **Efficiency** - Clean, minimalist aesthetic
3. **Performance** - Dynamic gradient and glow effects
4. **Accessibility** - Clear, recognizable at any size
5. **Professionalism** - Suitable for enterprise use

Black and white version for printing and limited-color applications.

## Usage Examples

### In Markdown
```markdown
![Aster Logo](assets/logo.svg)
```

### In HTML
```html
<img src="assets/logo.svg" alt="Aster Logo" width="200" height="200">
```

### As Favicon
```html
<link rel="icon" type="image/svg+xml" href="assets/favicon.svg">
```
