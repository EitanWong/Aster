🎨 Aster Logo 设计选项

请选择你最喜欢的设计风格

Option 1: Clean Star
简洁星形
简洁的 5 点星形，去掉了复杂的效果。

优点

  • 简单易识别
  • 经典设计
  • 易于缩放

缺点

  • 可能太传统
  • 科技感不足
Option 2: Letter A
字母 A
基于 Aster 首字母的现代设计，带有火花点。

优点

  • 独特易识别
  • 代表项目名
  • 现代感强

缺点

  • 科技感一般
  • 可能太简单
Option 3: Orbital
轨道
同心圆环设计，代表神经网络和推理过程。

优点

  • 科技感强
  • 代表 AI/推理
  • 现代专业

缺点

  • 可能有点复杂
Option 4: Triangle
三角形
向上的三角形，代表上升、速度和能量。

优点

  • 简洁现代
  • 有动感
  • 易于识别

缺点

  • 可能太简单
  • 科技感不足
Option 5: Tech Squares
科技方块
嵌套的旋转方块，现代科技风格。

优点

  • 专业科技感
  • 现代设计
  • 易于扩展

缺点

  • 可能有点冷
  • 不代表项目名
Option 6: Flower
花朵
6 瓣花朵设计，灵感来自 Aster 花。

优点

  • 优雅独特
  • 代表项目名
  • 易于识别

缺点

  • 科技感不足
  • 可能太柔和
