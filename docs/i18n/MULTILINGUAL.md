## 贡献翻译 / Contributing Translations

我们欢迎翻译贡献！如果您想帮助翻译文档：

We welcome translation contributions! If you want to help translate documentation:

### 步骤 / Steps

1. **选择文档** / Choose a document
   - 查看 [i18n.json](i18n.json) 了解翻译状态
   - Check [i18n.json](i18n.json) for translation status

2. **创建翻译** / Create translation
   - 复制英文版本 / Copy the English version
   - 翻译内容 / Translate the content
   - 保持格式和结构 / Maintain format and structure

3. **提交 PR** / Submit PR
   - 创建 Pull Request / Create a Pull Request
   - 描述翻译的语言和文档 / Describe the language and document translated
   - 等待审核 / Wait for review

### 翻译指南 / Translation Guidelines

- ✅ 保持原文的结构和格式 / Maintain original structure and format
- ✅ 保留代码块不变 / Keep code blocks unchanged
- ✅ 保留 URL 和文件路径 / Keep URLs and file paths unchanged
- ✅ 使用该语言的技术术语标准 / Use standard technical terminology for the language
- ✅ 翻译应该准确且自然 / Translations should be accurate and natural
- ❌ 不要翻译代码注释（除非特别需要）/ Don't translate code comments unless necessary
- ❌ 不要改变项目名称或品牌 / Don't change project names or branding
